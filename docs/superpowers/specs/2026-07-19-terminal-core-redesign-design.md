# Terminal-Core Redesign — Flapp

**Дата:** 2026-07-19
**Референс:** Ollama / Terminal-Core family (awesome-claude-design) — `#000 / #fff / mono`, жёсткие края, CLI-эстетика, grayscale-каркас, максимальный контраст, минимум украшений.

## Цель

Полностью перевести визуальный язык приложения с тёмной темы «Console» (сине-серые слоёные поверхности, лаймовый акцент `#C8F751`, насыщенные цвета категорий, скруглённые углы, sans-шрифт Geist) на **Terminal-Core**: чистый чёрный фон, белый текст, моноширинный шрифт везде, нулевые скругления, различие поверхностей рамкой, а не заливкой.

Смысловой (функциональный) цвет сохраняется, но переводится в **приглушённую, гармоничную с ч/б палитру** — так, чтобы категории/обложки оставались различимыми, а каркас читался как терминал.

## Решения (утверждены пользователем)

1. **Цвет категорий** — сохранить различимыми, но десатурировать в тон ч/б (dimmed-палитра). Не убирать полностью.
2. **Плеер** (`PlayerPage.tsx`) — тоже переводится в Terminal-Core; FL-скин (хром-панели, LCD) сносится.
3. **Оконные кнопки** (macOS-светофор в `TopBar`) — оставить цветными как есть (красный/жёлтый/зелёный).
4. **Волновая форма** — чисто белая на чёрном (без оттенка категории).

## Дизайн-система

### Каркас (строго моно / ч-б)

- **Фон:** `#000` для всего app-shell, панелей, полей ввода (well/sunken — чуть иной чёрный `#060606` максимум для разграничения, но в основном рамкой).
- **Поверхности:** различаются рамкой `1px`, не заливкой. Плоско.
- **Границы:** `--border-soft: #262626` → `--border-medium: #3a3a3a` → `--border-strong: #555`.
- **Текст:** `--text-strong: #ffffff` → `--text-body: #a3a3a3` → `--text-muted: #6b6b6b` → `--text-faint: #3d3d3d`. Нейтральный серый, без синевы.
- **Шрифт:** `Geist Mono` **везде**. `--font-sans` алиасится на mono, чтобы нативные `input/button/textarea/select` тоже стали моноширинными. Sans как отдельное семейство не используется.
- **Скругления:** все `--radius-*` = `0`. Terminal-Core угловат.
- **«Акцент»:** белый (`#fff`) как активное/выделенное состояние вместо лайма. `primary`-кнопка = инверсия: белый фон, чёрный текст (CTA как на ollama.com). Hover primary → лёгкое затемнение (`#e0e0e0`).
- **Фокус-ring:** `1px` сплошная белая рамка (box-shadow `0 0 0 1px #fff`), не размытое свечение.
- **Тени:** убрать у плоских поверхностей; сохранить лёгкую только у всплывающих (`--shadow-pop`).
- **Скроллбары, `::selection`:** серые, без цвета. Selection — инверсия (белый фон/чёрный текст) или серый.
- **Движение:** сохранить существующие длительности/ease; убрать цветовые переходы, где они завязаны на акцент.

### Функциональный цвет (dimmed)

Палитра категорий переводится в десатурированные терминальные тона, читаемые на чистом чёрном (ориентир — dimmed ANSI). Итоговые значения `--cat-*` (doc) и `GROUP_HEX` (categories.ts) приводятся к **единому** набору (сейчас они рассинхронизированы — токены насыщенные, GROUP_HEX уже землистый):

| Категория | dimmed hex |
|---|---|
| 808 | `#C77B5A` |
| Kick | `#C9915C` |
| Snare | `#C47385` |
| Clap | `#BB7E9E` |
| Hi-Hat | `#6FB3A6` |
| Open Hat | `#6FA3B8` |
| Perc | `#7E90B5` |
| Vox | `#A585B8` |
| FX | `#B87FA8` |
| Loop | `#8985B0` |
| Drum Loop | `#C2A15E` |
| Unsorted | `#7A7A7A` |

`--cat-*-bg` — тот же цвет с низкой альфой (`~14–1a`) только там, где нужна подложка; по умолчанию теги без заливки.

- **Тег категории** (`CategoryTag.tsx`): убрать цветную пилюлю → прямоугольник (radius 0) с рамкой `1px` в цвете категории, цветной точкой и белым/серым текстом. Цвет тонкий, каркас ч/б.
- **Волновая форма:** белая/светло-серая на чёрном; сыгранная часть ярче (`#fff`), несыгранная — тусклее (`#555`). Без оттенка категории.
- **Обложки:** остаются цветными (контент), но в рамке `1px`, radius 0.

### Плеер (снос FL-скина)

- Удалить/не использовать FL-токены `--chrome*, --work*, --ink*, --panel*, --btn(hi/lo), --groove, --lcd*, --rec` из `tokens.css` (временный блок FL-совместимости).
- Транспорт, таймлайн, piano-roll (`PlayerPage.tsx`, `MidiPianoRoll.tsx`, `MidiSection.tsx`, `useMidiPlayer.ts` — только визуал): чёрный фон, белые контролы, серые рамки, playhead белый, MIDI-ноты — dimmed-цвет категории/один нейтральный dimmed тон.

## Вне области

- Логика/данные/бэкенд не меняются — только визуальный слой.
- Функциональная перекомпоновка страниц (layout) не входит: меняется стиль, не структура навигации.
